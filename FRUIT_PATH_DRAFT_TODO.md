# Fruit Path Draft TODO

These candidates are derived from the current `forests` material, including the newer `Religion` and `Sensationalism` series.

They are meant to fill gaps not already covered by the existing fruit paths:

- `anxiety-trust`
- `burnout_grace`
- `control-surrender`
- `cynicism_hope`
- `emotional-reactivity_spirit-led-identity`
- `envy-gratitude`

## Strongest Next Drafts

- [ ] `Prayerlessness ↔ Communion-Based Dependence`
  `pathwayId:` `prayerlessness-communion`
  `Forest source:` `Naturalism`
  `Why it fits:` Naturalism normalizes prayerlessness that feels intellectually clean. This would name the habit of trusting only what can be measured, managed, or explained.

- [ ] `Conflict Avoidance ↔ Courageous Love`
  `pathwayId:` `avoidance-courageous-love`
  `Forest source:` `Moral Relativism`
  `Why it fits:` This directly matches "silence that feels kind" and "avoidance that feels peaceful." The truth-tree would help distinguish peacekeeping from truthful love.

- [ ] `Vagueness ↔ Convictional Clarity`
  `pathwayId:` `vagueness-conviction`
  `Forest source:` `Moral Relativism`
  `Why it fits:` The forest repeatedly trains people to mute conviction into niceness, self-edit constantly, and treat clear moral speech as cruelty.

- [ ] `Borrowed Certainty ↔ Trained Discernment`
  `pathwayId:` `borrowed-certainty-discernment`
  `Forest source:` `Religion`, with strong support from `Sensationalism`
  `Why it fits:` This comes straight out of "Discernment Cannot Be Outsourced." It would name the bad fruit of leaning on approved voices, systems, or charged moments instead of learning to test and attend before God.

- [ ] `Scrupulosity ↔ Reverent Freedom`
  `pathwayId:` `scrupulosity-reverent-freedom`
  `Forest source:` `Religion`
  `Why it fits:` The Ironwood normalizes fear of getting it wrong disguised as holiness. This is related to `control-surrender`, but it is more specifically about spiritual fear, conscience pressure, and a distorted picture of God.

- [ ] `Compliance ↔ Loving Obedience`
  `pathwayId:` `compliance-loving-obedience`
  `Forest source:` `Religion`
  `Why it fits:` This would trace outward obedience without inward tenderness back to rule-centered formation and move toward relational obedience grounded in love.

- [ ] `Experience Dependence ↔ Abiding Presence`
  `pathwayId:` `experience-dependence-abiding-presence`
  `Forest source:` `Sensationalism`
  `Why it fits:` Sensationalism trains people to need atmosphere, surge, or visible intensity in order to believe God is near. This is adjacent to `emotional-reactivity_spirit-led-identity`, but the core issue here is spiritual perception, not raw emotional volatility.

- [ ] `Charisma Trust ↔ Character Discernment`
  `pathwayId:` `charisma-trust-character-discernment`
  `Forest source:` `Sensationalism`
  `Why it fits:` "When Charisma Replaces Character" is already doing the worldview work. A fruit path here could help readers name why they keep trusting gifted force over tested holiness.

- [ ] `Urgency ↔ Patient Testing`
  `pathwayId:` `urgency-patient-testing`
  `Forest source:` `Sensationalism`
  `Why it fits:` This would address discernment weakened by pressure, speed, and the fear that questioning a spiritual claim means resisting God.

## Second-Wave Drafts

- [ ] `Flatness ↔ Worshipful Wonder`
  `pathwayId:` `flatness-wonder`
  `Forest source:` `Naturalism`
  `Why it fits:` Naturalism produces a life where awe gets replaced by analysis and hope gets reduced to probability. This would help name the lived experience of a closed, flattened world.

- [ ] `Boundarylessness ↔ Holy Love`
  `pathwayId:` `boundarylessness-holy-love`
  `Forest source:` `Moral Relativism`
  `Why it fits:` The forest makes boundaries feel unloving and moral lines feel unsafe. This path could show how love actually requires truthful limits.

- [ ] `Image Management ↔ Honest Repentance`
  `pathwayId:` `image-management-honest-repentance`
  `Forest source:` `Religion`
  `Why it fits:` The Ironwood trains polished behavior and managed appearances. This path would focus on confession, repentance, and telling the truth without curating an acceptable self.

- [ ] `Exhaustion Masked as Zeal ↔ Quiet Faithfulness`
  `pathwayId:` `zeal-exhaustion-quiet-faithfulness`
  `Forest source:` `Sensationalism`
  `Why it fits:` This would name the overdriven spiritual life that keeps calling depletion devotion. It overlaps with `burnout_grace`, so it is probably best treated as a later or more specialized draft.

## Notes On Overlap

- `control-surrender` already covers some of the control logic in `Religion`.
- `burnout_grace` already covers part of the exhaustion logic in `Religion` and `Sensationalism`.
- `emotional-reactivity_spirit-led-identity` already touches some sensation-heavy instincts, but not the specific theology of spectacle and urgency in `Sensationalism`.
- `anxiety-trust` overlaps some `Naturalism` and `Religion` fears, but does not yet cover prayerlessness, scrupulosity, or outsourced discernment directly.
